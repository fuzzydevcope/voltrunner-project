# VoltRunner
VoltRunner is a modular mobile power and monitoring system designed to support experimental electronics, survival engineering, and mobile living systems.

## Current Stage
Concept development and early prototyping.

## Goals
- Build a modular power system
- Integrate recycled electronics
- Develop monitoring interface
- Prototype physical structure

## Current Experiments
- Battery testing
- Digital frame display testing
- Hardware salvaging
- Cardboard prototyping

## Future Development
- Sensor monitoring
- Power management system
- Modular hardware design
- Software dashboard
